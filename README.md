# Dawn of the Eclipse - Character Calculator

If you're a reader of the 'Dawn of the Eclipse' Series, this program helps you to
understand the character values, or - even better - build your own characters in this
world!

Characters are saved as JSON files, and for your inspiration, one file is added as
a Sample - it's Alarac, the MC. This character sheet adheres to Alarics values during
Chapter 10 of the third book of the series, 'New Horizons'.

Just compile it and run it. 

If you supply a filename as argument when running the program, it will open the file name
as a character. During menu movements, you can use TAB key to cycle through your options.

Have fun building your own characters!
